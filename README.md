# Facial Expression Classification in Affective Computing for Early Mental Health Risk Detection
📌 Project Overview
This project focuses on leveraging Affective Computing and Machine Learning to classify human facial expressions. By analyzing emotions through the FER-2013 dataset, the system aims to provide a technological foundation for early detection of mental health risks, such as depression and anxiety, through non-verbal cues.

🚀 Key Features
- Facial Expression Recognition (FER): Categorizing emotions into 7 basic states (Angry, Disgust, Fear, Happy, Sad, Surprise, Neutral).

- Deep Learning Models: Implementation and comparison of various architectures (CNN, ResNet, etc.) to optimize accuracy.

- Mental Health Application: Establishing a link between prolonged negative emotional states and potential mental health issues.

🛠 Tech Stack
- Language: Python

- Libraries: TensorFlow/Keras, OpenCV, Pandas, NumPy, Matplotlib

- Dataset: FER-2013 (Facial Expression Recognition 2013)

# Ứng Dụng Phân Lớp Biểu Cảm Khuôn Mặt Nhằm Phát Hiện Sớm Rủi Ro Sức Khỏe Tâm Thần
📌 Tổng quan dự án
Dự án này tập trung vào việc ứng dụng Điện toán cảm xúc (Affective Computing) và Học máy để phân loại các biểu cảm khuôn mặt người. Thông qua việc phân tích cảm xúc từ bộ dữ liệu FER-2013, hệ thống hướng tới việc xây dựng cơ sở công nghệ hỗ trợ phát hiện sớm các rủi ro về sức khỏe tâm thần như trầm cảm hay lo âu dựa trên các tín hiệu phi ngôn ngữ.

🚀 Các tính năng chính
- Nhận diện biểu cảm (FER): Phân loại cảm xúc thành 7 trạng thái cơ bản (Tức giận, Ghê tởm, Sợ hãi, Hạnh phúc, Buồn bã, Ngạc nhiên, Trung tính).

- Mô hình học sâu: Triển khai và so sánh các kiến trúc mạng (CNN, ResNet,...) để tối ưu hóa độ chính xác.

- Ứng dụng sức khỏe tâm thần: Thiết lập mối liên hệ giữa các trạng thái cảm xúc tiêu cực kéo dài và các vấn đề tâm lý tiềm ẩn.

🛠 Công nghệ sử dụng
- Ngôn ngữ: Python

- Thư viện: TensorFlow/Keras, OpenCV, Pandas, NumPy, Matplotlib

- Bộ dữ liệu: FER-2013
